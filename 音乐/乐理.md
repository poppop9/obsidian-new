# 和弦
>[!quote] 和弦
>和弦 就是由<u>三个，或者三个以上的音</u>组成的一组声音

- 一级和弦 C --->>> 1 3 5
- 二级和弦 Dm --->>> 2 4 6
- ……
- 七级和弦 B --->>> 7 2 4

## 丰富和弦
### 调整节奏
- **一拍** : （x x x x）
	- 平稳感
- **两拍** : 本来是 1s 一拍（x x x x） --->>> 我现在可以 1s 两拍（x x x xx），一般加的那一拍是和弦内音，~~比如一级和弦，加的就是低音5~~
	- 推动感
- **四拍** : 1s 一拍 --->>> 1s 四拍（x x x xxxx），~~比如一级和弦，就是 5 3 1 低音5~~
	- 会有一种爆发感，适合用在进副歌

>[!quote] 和弦内音
>和弦内音 就是在和弦里面的音
>- 和弦内音加进来是不会影响听感的
>- 只要弹和弦内的音，节奏正确，随意变化，都可以

### 和弦把位
>[!quote] 和弦把位
>和弦把位 就是把和弦的第一个音换上去，~~比如一级和弦，就把 1 换成 高音1，还可以继续换，把 3 换成 高音3~~
