# 和弦
>[!quote] 和弦
>和弦 就是由<u>三个，或者三个以上的音</u>组成的一组声音。同一个和弦又可以分为：
>- 大三和弦：大三度 + 小三度 --->>> 色彩明亮，有很好的稳定性，常作为乐曲的终止和弦
>- 小三和弦 m：小三度 + 大三度 --->>> 色彩暗淡，emo，也很稳定，但是不及大三和弦
>- 增三和弦 aug：大三度 + 大三度 --->>> 色彩诡异，不稳定
>- 减三和弦 dim：小三度 + 小三度 --->>> 色彩非常诡异，不稳定，效果刺耳
>

在三和弦里，第一个音叫根音，第二个音叫做三音（与根音差了三度），第三个音叫做五音（与根音差了五度）
- 一级和弦 C --->>> 1 3 5
- 二级和弦 D --->>> 2 4 6
- ……
- 七级和弦 B --->>> 7 2 4

## 丰富和弦
### 调整节奏
- **一拍** : （x x x x）
	- 平稳感
- **两拍** : 本来是 1s 一拍（x x x x） --->>> 我现在可以 1s 两拍（x x x xx），一般加的那一拍是和弦内音，~~比如一级和弦，加的就是低音5~~
	- 推动感
- **四拍** : 1s 一拍 --->>> 1s 四拍（x x x xxxx），~~比如一级和弦，就是 5 3 1 低音5~~
	- 会有一种爆发感，适合用在进副歌

>[!quote] 和弦内音
>和弦内音 就是在和弦里面的音
>- 和弦内音加进来是不会影响听感的
>- 只要弹和弦内的音，节奏正确，随意变化，都可以

### 和弦把位
>[!quote] 和弦把位
>和弦把位 就是把和弦的第一个音换上去，~~比如一级和弦，就把 1 换成 高音1，还可以继续换，把 3 换成 高音3~~
>
>- 以中音 c 为中心，转位到低音 so 与中音 so 之间的区域合适


# 即兴伴奏
选择伴奏的音区很重要，比如如果人唱歌在中音区到高音一区，你伴奏也是在这个区域，那就不和谐，**我们伴奏应该给主奏留出空间，到别的音区去配合主奏**
