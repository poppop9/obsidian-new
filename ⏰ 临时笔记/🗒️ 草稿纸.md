
# 编程路线
RPC ：一种进程间的通信方式，它实现了轻松地允许一个进程调用另一个进程的服务

API 网关 ：一种设计风格和架构原则，用于创建可伸缩、轻量级、可维护和可扩展的网络服务，它承接 http 请求，调用 RPC 服务，并负责对外部请求的协议转换、参数校验、鉴权、切量、熔断、限流、监控、风控等各类共性的通用服务
