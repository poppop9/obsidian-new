# ❤️ if
- if
```go
if v < 1000 {
	fmt.Printf("v小于1000\n")
}
```
- if … else …
```go
if v < 1000 {
	fmt.Printf("v < 1000\n")
} else {
	fmt.Printf("v > 1000\n")
}
```
- if … else if … else …
```go
if(condition_1) {

} else if(condition_2) {

} else {

}
```
