# ❤️ 边框
- 【部分边框】
	- `border-t` 顶部边框
	- `border-r` 右侧边框
	- `border-b` 底部边框
	- `border-l` 左侧边框
- 【边框颜色】 `border-red-500` 
- 【边框宽度】 
	- `border-0` 无边框
	- `border-2` 2px 宽的边框


# ❤️ 圆角
- 【无圆角】
	- `rounded-none` 取消圆角
	- `rounded-tl-none` 左上无圆角
	- `rounded-tr-none` 
	- `rounded-bl-none` 
	- `rounded-br-none`
