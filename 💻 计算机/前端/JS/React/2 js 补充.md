# ❤️ 数组 / 集合
<u>查找</u> ：
- `indexOf(元素)` 跟据元素返回数组索引

---

<u>流操作</u> ：
- `map(lambda 表达式)` 转换

```jsx
var index = awardList.current.map(item => item.awardId).indexOf(awardId);
```

# ❤️ JSON
- `JSON.stringify(Object)` Object -> String
- `JSON.parse(json字符串)` json 字符串 -> Object
