# 💡 核心观点
我们常以为自己能同时专注于多件事，但对于大多数需要认知参与的任务来说，大脑并不是在“并行处理”，而是在不同任务之间快速切换

这种切换并非没有成本。每次从一个任务转向另一个任务时，大脑都需要重新定位目标、规则和上下文
- 时间损耗：重新进入状态需要时间
- 认知负担增加：工作记忆被频繁占用
- 错误率上升：更容易遗漏细节或判断失误
- 疲劳感增强：注意力不断被拉扯，主观上更累

因此，多任务处理带来的往往不是更高效率，而是「忙碌感」大于「产出」

# 📌 示例
心理学研究曾比较两组人在任务执行中的表现
- 一组需要同时处理多项任务
- 另一组一次只专注完成一项任务

结果显示，多任务组虽然主观上更容易认为自己效率很高，但客观表现通常更差，他们完成任务的速度更慢、错误更多、结果质量更低，整体表现不如单任务组。

---

日常生活中的例子
```
一边回消息一边写报告，结果写作思路频繁中断

一边开会一边处理邮件，最后两边的信息都吸收不完整

一边学习一边刷短视频，学习内容更难进入长期记忆
```

# 🪴 延伸
- 专注不一定要用强大的意志力去逼迫自己，只要你进入心流，你会极度专注，但一点都不觉得累
