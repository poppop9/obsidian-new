# 📚 Non-Neediness
> 需求感 Neediness 是指一个人更重视他人对自己的看法，而不是自己的看法。一个男人的吸引力与他有多需要别人成反比（他越不需要别人，他对女人来说就越有吸引力）

🔴 欲望 vs. 需求 ：Non-Neediness 并不是无欲无求
- 无需求感 = 我有欲望 + 我采取行动 + 希望有结果 + 不依赖结果
- 有需求感 = 我有欲望 + 我采取行动/不敢行动 + 希望有结果 + 极度依赖结果

🔴 表现形式 ：
- 关于关注 ：一个专注于自己的事、不乞求别人关注的男人，反而显得神秘、高价值
- 关于被喜欢 ：如果你害怕别人不喜欢你，你就会变得小心翼翼、虚伪。但如果你心态是：你不喜欢我也没关系，我接受，这种真实和松弛感，反而会让人觉得你很酷，从而更喜欢你
- 关于意见分歧 ：当你拼命想说服别人时，会激起对方的防御心理，让对方更想反驳你。但如果你很自信地表示没关系，我尊重你的看法，对方反而会放下戒备，开始思考你的观点，甚至反过来认同你

🔴 Non-Neediness 的边界 ：
- 有魅力的独立 ： 我不需要你来证明我很棒，但我真心欣赏你，愿意和你建立深层的联系
- 有毒的自恋 ： 我只爱我自己，我和你在一起只是为了利用你来证明我很棒

🔴 有毒的自恋 ：自恋男（极度自我中心、缺乏同理心、需要不断的赞美）可以吸引到两种类型的女人 ：
- 自恋的女性（同样自我中心、精于算计） ：这是一种交易型关系，并没有真正的爱，只有价值交换。自恋女选择自恋男，是因为这个男人看起来很光鲜、有地位或有魅力，这能让她看起来也很高级。他们爱的不是对方，而是对方眼中的自己。
- 低自尊的女性（极度缺乏安全感、内心空虚、甚至有受虐倾向或情感创伤史） ：她通过依附于一个看似伟大的男人，来让自己感觉也变得伟大，为了维持这种虚幻的连接，她愿意忍受对方的羞辱和恶劣对待

# 📚 Vulnerability
> 脆弱性 并非软弱，而是敢于在结果未知的状况下，毫无保留地展示真实的自己（不仅仅意味着愿意分享你的恐惧或不安全感，还意味着让自己处于可能被拒绝的位置），这是一种自信和高地位行为

- 信任感的来源 ：女性在情感和身体上只能信任一个真实的人，而非一个完美的假面，学会接受不完美，她会被你的真实所吸引
	- 完美是假的 ：总是能说出完美情话的男人是不会被她们信任的，完美意味着他在掩饰，他在试图操控印象，这意味着他需要对方的认可
	- 棱角且真实 ：一个既有优点又有缺点，并且能够承认后者，嘲笑前者的男人，这是一个她可以信任的、可以敞开心扉的男人

- 建立筛选机制： 那些因为你展示真实而被吓跑的女人，本来就不适合你，留下来的，是真正欣赏你的人

🔴 脆弱性的行为 ：讲一个可能不好笑的笑话、坚持一个可能会冒犯别人的观点、向一群你不认识的人介绍自己、告诉一个女人你喜欢她，想和她约会

🔴 行为指南 ：
- 拒绝交易式社交，而是建立真实联系 ：不要试图收买别人的喜欢，你不能通过送礼物、请吃饭或无脑夸赞来换取对方的爱，这只会让你看起来像个试图推销假冒伪劣产品的推销员，只有当你真正花时间了解对方，因其特质而表达欣赏时，这种欣赏才有效

- 残酷的诚实 ：有时，诚实可能会伤人，但这种残酷而愤怒的诚实往往比恭维更能让女性兴奋。例如你愿意打断一个女人，告诉她越界了，当你告诉一个女人你生活中可以容忍什么，不能容忍什么，这其实是在向她传达你最有吸引力的元素。这比一个有趣的故事或游戏要有用得多

> 发展一种让自己快乐、丰富的生活方式，过自己想过的生活
