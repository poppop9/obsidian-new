# 📚 课题 I - 自我认知
🏷️ 生活方式的定义 ：生活方式 = 某人如何看世界 + 又如何看自己  + 如何给这些赋予意义，生活方式从狭义上来讲，可以理解为性格；从广义上来说，包含了某人的世界观或人生观

🏷️ 生活方式的形成 ：阿德勒认为大约是在 10 岁左右的时候，自己无意识地选择了自己的性格，更大点讲叫世界观

🏷️ 你维持生活方式的真相 ：缺乏勇气
- 抱怨不幸，实际上是因为保持不幸比较轻松
- 改变的成本很高，你要面对未知的恐惧，和承担失败的风险

🏷️ 自我认知的核心主张 ：你的人生由你自己选择，而非被过去决定
- 阿德勒心理学明确否定心理创伤，而弗洛伊德的心理创伤学说认为心灵过去所受的伤害是引起目前不幸的罪魁祸首
- 虽然外部因素会对你的人生产生很大的影响，但是任何经历本身并不是成功或者失败的原因，你的人生还是由你自己选择的，是你自己选择如何生活
- 我们并非因心理创伤而痛苦，而是为了达成某种目的而创造了这些情绪（~~例如与异性交谈时脸红，这并非过去的阴影所致，而是因为你心生恐惧，便制造出‘脸红’这一症状作为逃避的借口，以此自我安慰：‘我只是因为脸红才做不到，只要不脸红，我也能行~~）

# 📚 课题 II - 人际关系
即使人们有各种不满，但还是认为保持现状更加轻松、更能安心，害怕未知，改变需要很大的勇气

如果有人骄傲自大，那一定是因为他有自卑感

健全的自卑感不是来自与别人的比较，而是来自与“理想的自己”的比较
  
之所以有很多人虽然取得了社会性的成功，但却感觉不到幸福，就是因为他们活在竞争之中。因为他们眼中的世界是敌人遍布的危险所在
  
无法真心祝福过得幸福的他人，那就是因为站在竞争的角度来考虑人际关系，把他人的幸福看作我的失败，所以才无法给予祝福

# 📚 杂
通常我们认为批评会打击信心，所以要多表扬，表扬能让人更自信。但阿德勒认为：表扬和批评，本质上是一回事。它们都是糖衣炮弹
- 表扬背后隐藏着不平等 ：当你表扬别人时，你潜意识里没把孩子当成一个独立、平等的个体，而是在评估他的表现
	- 批评 = 我觉得你不行，我要纠正你（俯视视角）
	- 表扬 = 我觉得你做得不错，我要奖赏你（依然是俯视视角）
- 表扬的真正目的是操纵 ：表扬通常是为了让对方下次继续做这件事
	*   “你把玩具收起来了，真棒”（下次也要乖乖收玩具，别给我添麻烦）
	*   “你考了100分，太聪明了”（你要继续考高分，满足我的期待）
- 表扬会剥夺对方的勇气 ：会导致对方上瘾，过度依赖外部评价，总是向外寻求认可，缺乏自我价值感

那如果不批评也不表扬，应该怎么办 ：把对方看作一个平等的伙伴，建立横向关系，进行鼓励和表示感谢，对方会觉得自己有贡献，有价值
- 表扬（评价）：你帮妈妈洗碗了，真棒！真是个好孩子！
* 鼓励/感谢（表达主观感受）：你帮妈妈洗碗了，谢谢你，帮了我大忙了，我很开心

人只有在觉得自己对共同体有用的时候，才能体会到自己的价值
