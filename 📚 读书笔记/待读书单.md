# 表达
1. 《关键对话》（Crucial Conversations: Tools for Talking When Stakes Are High）  
    作者：Patterson, Grenny 等四人  
    这本书是沟通类书籍的“天花板”，教你在高压、高风险、高情绪场景下依然能好好说话（跟老板谈加薪、跟伴侣吵架、跟父母谈钱，都适用）
2. 《说话之道》（Crucial Accountability / 原名 Crucial Confrontations）  
    算是《关键对话》的进阶版。专门教你如何处理别人违背承诺、不负责任、踩你底线的情况，既能指出问题，又不伤感情。
3. 《金字塔原理》（The Pyramid Principle）  
    作者：巴巴拉·明托，教你如何让你的表达“结构化、逻辑清晰、重点先行”。写邮件、做汇报、写方案、甚至日常聊天，都能瞬间显得很专业。
4. 《如何阅读一本书》  
    作者：莫提默·艾德勒，看似和表达无关，但其实是输入的最高阶方法。你读得越深、越有体系，你的表达才会真正有深度。所有真正会表达的人，背后一定是个阅读高手。
5. 《故事思维》（Storynomics / 或侯虹斌《故事力》）  
    想让人记住你说的话？用讲故事。人类大脑对故事的记忆是普通信息的22倍。这本书教你把任何内容都包装成让人上瘾的故事。
6. 《说话就是生产力》- 刘松妍（国内最顶尖的表达教练）
7. 《声音训练手册》- 克里斯汀·林克莱特（演员专业用书）
8. 《即兴演讲》- 崔西（世界即兴演讲冠军教你脱稿说话）
