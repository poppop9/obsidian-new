## 📖 四气（寒热）
**寒、凉、平、温、热。判断标准很朴素，不是看颜色，不是看产地，而是古人大量食用后，观察身体的实际反应，经过几千年积累总结出来的**

> [!NOTE]
> **为什么有 5 种特性，但叫做四气**
> 
> 最早《神农本草经》只提寒、热、温、凉。平性是后来补充的

▶ **四气可以被改变**
- 加热烹饪 —— 凉性的白萝卜煮熟后寒性减弱
- 发酵 —— 牛奶发酵成酸奶后偏温
- 加盐 —— 白萝卜腌制成腌萝卜后偏温
- 加糖 —— 生姜本温，但姜糖水的热性会被糖中和一部分
- 晒干/烘干 —— 鲜姜晒成干姜后偏热

▶ **同一食物不同部位可能性质不同**

> 橘子肉凉性，陈皮温性
> 
> 生姜皮凉性，生姜肉温性

### 📝 热
大力驱寒、温通经络、振奋阳气、容易上火
- 适合 —— 寒证严重的人（手脚冰凉、面色苍白、腹泻清稀）
- 过用 —— 口干舌燥、便秘、流鼻血、长疮

> 辣椒、花椒、肉桂、干姜、白酒、附子、肉桂

### 📝 温
温和地暖身、补气血、助消化
- 适合 —— 体质偏寒、阳气不足的人

> 生姜、大葱、韭菜、大蒜
> 
> 羊肉、鸡肉、鹿肉
> 
> 糯米、红枣、桂圆、荔枝、核桃
> 
> 红茶、黄酒

### 📝 平
性质平和，寒热偏性不明显，适合大多数人长期食用

> 大米、小麦
> 
> 玉米、胡萝卜、山药、土豆、红薯、花生、黄豆、黑豆
> 
> 猪肉、牛肉、鸡蛋、鲤鱼、鲫鱼	
> 
> 苹果、葡萄

> [!NOTE]
> **平性不等于没有功效，只是不会明显改变身体的寒热状态**

### 📝 凉
轻度清热、缓解轻微的热证
- 适合 —— 体质偏热但不严重的人

> 薏米、绿豆、豆腐、白萝卜、芹菜、菠菜、茄子
> 
> 鸭肉
> 
> 梨、柑橘、草莓、芒果
> 
> 绿茶、菊花茶

### 📝 寒
大力清热、泻火、解毒、凉血
- 适合 —— 实热证明显的人（发烧、口臭、便秘、面红目赤）
- 不适合 —— 脾胃虚寒的人不宜多吃
- 过用 —— 腹泻、腹痛、手脚冰凉、消化不良

> 西瓜、香蕉、柿子、猕猴桃
> 
> 苦瓜、黄瓜、冬瓜、海带
> 
> 螃蟹、蛤蜊
> 
> 凉茶

## 五味
**五味酸、苦、甘、辛、咸，判断标准来自古人长期食用后观察到的身体反应，而不仅仅是舌头尝到的味道**


```
        苦 → 心（火）
         ↗
酸 → 肝（木）    甘 → 脾（土）
         ↘
        辛 → 肺（金）
         
        咸 → 肾（水）
```

### 酸（肝、胆）
**酸的本质是往内收、往里固。就像你咬一口柠檬，整个人会"缩"起来**

|          | 说明                   |
| -------- | -------------------- |
| **功效**   | 收敛止汗、固涩止泻、生津止渴       |
| **适合**   | 多汗、慢性腹泻、遗精遗尿、口干      |
| **不合适**  | 胃酸过多                 |
| **典型食物** | 醋、柠檬、山楂、乌梅、石榴、酸枣、五味子 |

酸味过多 → 肝气收敛太过 → 木太多了就会克制脾土

> 《黄帝内经》："多食酸，则肉胝䐢而唇揭"，吃太多酸的，皮肉会变粗糙，嘴唇会干裂翘皮

### 苦（心、小肠）
**苦的方向是向下、向外排，泻心火**

|          | 说明                           |
| -------- | ---------------------------- |
| **功效**   | 清热、泻火、解毒、燥湿、降逆（止呕止咳）         |
| **适合**   | 实热证、上火、口腔溃疡、湿气重              |
| **不适合**  | 脾胃虚寒                         |
| **典型食物** | 苦瓜、莲子心、苦菊、芹菜叶、茶叶（尤其绿茶）、陈皮、杏仁 |

过量的后果：泻心火太过，火生不了土，伤脾胃阳气

《黄帝内经》："多食苦，则皮槁而毛拔"
（吃太多苦的，皮肤干枯，毛发脱落）

### 甘（脾、胃）
甘本质是补益、和中、缓急

|          | 说明                               |
| -------- | -------------------------------- |
| **功效**   | 补气血、健脾胃、缓解疼痛和痉挛、调和药性             |
| **适合**   | 气血虚弱、脾胃虚、疲劳乏力、疼痛拘急               |
| **不适合**  |                                  |
| **典型食物** | 大米、小麦、山药、红枣、蜂蜜、南瓜、红薯、桂圆、枸杞、牛肉、鸡肉 |

过量的后果：甘味过多 → 脾被壅滞（塞满了消化不动）→ 生痰生湿
→ 同时克制肾水 → 骨骼问题、脱发

《黄帝内经》："多食甘，则骨痛而发落"
（吃太多甘甜的，骨头疼痛，头发脱落）

日常例子：

生病体虚时喝小米粥、吃山药 → 甘味补脾益气
但天天吃甜食 → 痰湿壅盛、发胖、长痘

|        | 甘温       | 甘寒/甘凉      |
| ------ | -------- | ---------- |
| **作用** | 补气补阳     | 滋阴生津       |
| **例子** | 红枣、桂圆、羊肉 | 梨、甘蔗、百合、银耳 |

## 📖 碳水
### 📝 按消化吸收速度分
**按消化吸收速度分，分为快碳、慢碳。划分标准比较粗糙，和很多因素有关**
- 加工程度
- 膳食纤维量
- 蛋白质、脂肪搭配
- 烹饪方式
- 食物成熟度
- 食物颗粒完整度

> 白米饭通常比糙米饭更“快”
> 
> 燕麦片通常比即食燕麦粉更“慢”
> 
> 土豆本身不一定特别“快”，但做成土豆泥可能更快
> 
> 整颗水果比果汁更慢

> [!NOTE]
> **严格来说，快碳、慢碳不是标准的营养学正式分类，而是健身圈、减脂圈的通俗说法**

▶ **快碳** —— 消化吸收快，吃完血糖上升较快

> 白米饭、精白面粉、白面条、白馒头

▶ **慢碳** —— 消化吸收相对慢，血糖上升更平稳，饱腹感通常更强

> 全谷物、薯类、玉米、土豆、南瓜、燕麦、芋头

### 📝 按营养质量分
**按营养质量分，分为优质碳水、精制碳水，衡量标准有很多**
- 膳食纤维
- 维生素
- 矿物质
- 加工程度高不高
- 饱腹感
- 对血糖稳定是否更友好
- 是否容易吃过量

▶ **优质碳水** —— 天然、少加工、富含纤维和微量营养素的碳水

> 全麦面包、紫薯、杂粮粥、玉米、荞麦面、糙米饭、山药、土豆、红薯、燕麦、南瓜、芋头、小米粥、薏米、莲藕

▶ **精制碳水** —— 精制程度高、纤维少、容易过量的碳水

相生：
木 → 火 → 土 → 金 → 水 → 木

相克：
木 → 土 → 水 → 火 → 金 → 木

木生火：木材燃烧产生火
火生土：东西燃烧后化为灰烬，归于土
土生金：金属、矿物蕴藏于土中
金生水：古人认为金属表面会凝结水珠，因此说金生水
水生木：水滋养树木生长

肝(木)生心(火)
心(火)生脾(土)
脾(土)生肺(金)
肺(金)生肾(水)
肾(水)生肝(木)
