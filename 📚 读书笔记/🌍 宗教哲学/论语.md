# 📚 孔子介绍
孔子姓孔，名丘，字仲尼，仲表示他在兄弟中排行第二，死于鲁哀公十六年，即公元前 479 年，终年实七十二岁

周武王灭了殷商，但商朝的王室后人还在，商朝的百姓还在。如果完全消灭他们，不仁不义；如果放任不管，他们可能造反。所以周武王给了商纣王的庶兄（同父异母的哥哥）微子启一块地，就封他到宋地（今河南商丘一带），建立了宋国，而孔子就是微子启的后人


- 孔氏的由来 ：孔子的先祖孔父嘉是宋国宗室，因为距离宋国始祖已经超过五代（五世亲尽，要另立氏），便改为孔氏
- 从宋国到鲁国 ：华氏家族在宋国势力强大，孔父嘉的后人长期受到威胁，防叔（孔父嘉的后代）为避祸而逃奔到鲁国
- 孔子的直系世系 ：防叔 → 伯夏 → 叔梁纥 → 孔子

殷商是奴隶社会，崇尚鬼神、重视祭祀

卜辞就是甲骨文

孔子从很卑微的小吏，如'委吏'（仓库管理员），如'乘田'（主持畜牧者），进而受到鲁国权臣季氏的赏识，才进入'大夫'的行列

孔子的政治实践 —— 在鲁国行不通，到齐国也碰壁，到陈蔡等小国，更不必说了。在卫国，被卫灵公供养，住了较长时间，晚年回到鲁国专心于教育和整理文献

孔子大半辈子精力用于教育和整理古代文献，他对后代的最大贡献也就在这里
- 教育 —— 创办私学、培养弟子、确立教学方法
- 整理、编订了六经 —— 诗经、尚书、礼记、乐经、易经、春秋



## 孔子思想体系的渊源
**孔子的思想渊源是复杂的、多方面的 —— 底层生活经历的烙印，鲁国特殊国情的塑造、广博学习古代文献的积淀、春秋时代众多杰出人物思想的影响。因此对于孔子不能简单地"肯定一切或否定一切"**

🔴 **出身与早年经历的影响** —— 孔子虽为殷商后裔，但到他这一代已沦为平民。他自述"吾少也贱"，幼年丧父，由寡母抚养长大，为谋生不得不从事各种杂活，如做过管理仓库的"委吏"和管理牲畜的"乘田"。孔子自幼与贫贱之人共事，这些底层社会的经历和人际交往不可能不对他的思想产生影响，所以孔子的思想中带有对平民生活的切身体验和理解

🔴 **鲁国国情的影响** —— 这种"有改革而改革又不彻底"的国情，直接让孔子也既有革新的一面，又有保守的一面
- 鲁国政权掌握在季、孟、仲孙三家手中，季氏势力最大，季氏做法呈现出矛盾性 ：一方面推行封建军赋制度、另一方面却不敢取代鲁君自立
- 季氏矛盾原因 ：当时尚无权臣取代君位的先例，鲁国又是弱国，怕招来大国攻伐；鲁国作为周公后代的封国，周礼传统力量仍然强大

🔴 **春秋时代杰出人物的影响** —— 孔子对当时和此前的政治家、思想家有着深入的了解和明确的评价，这些人物的言行，或多或少地影响了孔子思想的形成
- **子产** —— 孔子极为赞扬。子产不毁"乡校"、允许民间议政的有限民主作风，被孔子以"仁"来评价，体现了孔子对开明政治的认同
- **臧文仲** —— 被孔子批评为"三不仁"、"三不智"，其中包括压抑贤才柳下惠一事，反映出孔子"用人唯贤"的主张
- **公叔文子** —— 因大力提拔贤才而受到孔子赞许
- **叔向** —— 孔子对他没有什么议论，作者推测可能因为叔向政治态度过于保守

## 孔子论天、命、鬼神、卜筮
**孔子是清醒的理性主义者、唯物主义者，其治学严谨导致他对超自然现象持怀疑主义，而非宿命论或鬼神崇拜**

🔴 **孔子对“天”的态度** —— 孔子称"天"多为情绪语言，未必真信有意志的天

🔴 **孔子对“命”的态度** —— 孔子谈"命"皆针对人事困惑，非宇宙论

🔴 **孔子对“鬼神”的态度** —— 孔子不信鬼神实有，但利用殷人重鬼神的传统为现实伦理服务，通过丧葬祭祀仪式，培养民众的忠厚品德，维护统治秩序

🔴 **孔子对“卜筮”的态度** —— 孔子明确否定占卜，认为人的命运和吉凶主要取决于自身的行为和道德，而非外在的占卜结果，强调通过自身努力来改善处境，而不是依赖卜筮来预知未来

## 孔子的政治观和人生观
在春秋时代，除了郑国子产等少数世卿有心救世外，出身下层而有志于救世的人物中，没有人能与孔子相比

🔴 **孔子时代的政治形势** —— 要实行较大的政治变革，没有适当的力量凭借是不可能的，孔子空有大志，只能感叹而终
- **周王室** —— 已经无法恢复权力和威信，这是人尽皆知的事实
- **霸主时代** —— 齐桓公、晋文公那样的霸主已成陈迹
- **各国内乱** —— 中原各国不是政权落入卿大夫之手，就是家臣掌握国家政权。晋国由六卿相争到只剩韩、赵、魏和知伯四家
- **鲁国** —— 国君失去权力已经五代，政权落入大夫之手也已四代，三桓（季孙、孟孙、叔孙）的后代虽然势力渐微，但已处于"陪臣执国命"的局面
- **其他国家** —— 卫、陈、蔡等国小力微；秦国自穆公、康公之后无力过问中原；楚国被吴国打得精疲力尽

🔴 **政治主张**
1. **治国三要素** —— 粮食充足、军备完善、人民信任（最为重要）
2. **正名主张** —— "君君，臣臣，父父，子子"，就是君主要像君主的样子，臣子要像臣子的样子，父亲要像父亲的样子，儿子要像儿子的样子

🔴 **孔子的人生观** —— 既有远大的政治抱负，又保持着朴素正直的生活态度
- "发愤忘食，乐以忘忧，不知老之将至" —— 发愤学习和工作时忘记吃饭，快乐时忘记忧愁，不知道衰老即将到来
- "不义而富且贵，于我如浮云" —— 不合道义的富贵，对孔子来说如同浮云，说明孔子能够过穷苦生活

## "忠恕"与"仁"
"忠"的本义是"中心" —— 心在中间，不偏不倚，真诚无伪
"恕" 就是"推己及人"——拿自己的心去体会别人的心

> "己所不欲，勿施于人" —— 我不喜欢的事，我也不对别人做
> "己欲立而立人，己欲达而达人" —— 我想要的好东西，我也帮别人得到

仁 = 忠 + 恕 = 时时处处替别人着想

## 孔子对后代的贡献
- 整理与传授古代文献 —— 孔子之前已有不少文献典籍，孔子一方面学习研究，一方面加以整理，同时向弟子传授
- 孔门弟子及其文化传承 —— 孔子的弟子实际上不过七十多人。《左传》多记载子路、冉有、子贡的言行。后辈学生中以子游、子夏、曾参最为著名，他们不做官，多半从事教学
- 孔子的学习态度 —— 孔子虽说过"生而知之者上也"，意思是生下来就懂得道理的人，是最上等的。自己却说："我非生而知之者，好古，敏以求之者也"，意思是我不是生来就懂的人，我只是爱好古代文化，勤勉努力去追求学习的人
- 孔子的教学方法
	- 孔子自称"诲人不倦"，意思是不厌其烦地教导学生，而且对学生毫无隐瞒。过去不少工艺师傅和拳术教师对学生总要留一手，不愿把全部本领传授出来。孔子则对学生无所保留，因而赢得学生无限的尊敬和景仰
	- "因材施教"，孔子对每个学生都非常了解，并针对不同学生做出不同的评价和指导。即使同一个问题，因提问者不同，回答也不同。颜渊、仲弓、司马牛三人"问仁"，孔子给出三种不同的答案。甚至子路和冉有都问"闻斯行诸"（听到道理就该马上实行吗），孔子的回答竟完全相反
	- "不愤不启，不悱不发"，意思是不到学生苦思冥想而想不通时不去开导，不到学生想说却说不出来时不去启发
- 孔子开创私学的历史功绩 —— 孔子以前，学术教育掌握在官府手中。私人设立学校、开门招生，而且学费非常低廉，从古至春秋时代，孔子恐怕是第一人，孔子的学生中出身贫贱的居多

# 📚 论语背景
🔴 《论语》命名的意义 —— "论" ＝ 论纂（编辑整理），"语" ＝ 语言（孔子及弟子的言语），"论语" ＝ 把从孔子那里听来的话编纂整理起来

🔴《论语》并非一人一时之作，而是由孔子的弟子和再传弟子在不同时期分别记录，最终由曾参（曾子）的学生在战国初期编定成书的
- 《论语》中存在大量重复现象，唯一合理的解释是孔子的言论在当时由不同弟子各自记录，后来汇集在一起才形成了这部书
- 部分章节出自孔子弟子本人之手
- 部分章节出自再传弟子之手
- 著作年代有先有后，《论语》各篇章的写作时间前后相距可能不止三五十年
- 最终编定者是曾参的学生 —— 曾参的言行记载在所有孔子弟子中最多；《论语》记载的最晚事件指向曾参晚年

《论语》的着笔开始于春秋末期（约公元前 5 世纪上半叶），编辑成书则在战国初期（约公元前 5 世纪下半叶至前 4 世纪初）

🔴 论语的版本
- 论语流传到汉代时，存在三个不同版本
	- 鲁论语：二十篇，是鲁地学者传授的版本
	- 齐论语：二十二篇，比鲁论多出《问王》和《知道》两篇，其余二十篇的章句与鲁论大体相同
	- 古文论语：二十一篇，没有《问王》《知道》，但将《尧曰》篇中"子张问"一段独立分出，形成两个《子张》篇。篇次与前两种不同，文字差异达四百余字
- 张侯论 ：张禹先学鲁论，后学齐论，将二者融合为一，内容"采获所安"（取自己认为合理的部分），我们今天通行的论语，基本上就是张侯论

## 📖 学而篇
> **子曰：学而时习之，不亦说乎？有朋自远方来，不亦乐乎？人不知，而不愠，不亦君子乎？**
> 
> 孔子说：学了，然后按一定的时间去实习它，不也高兴吗？有志同道合的人从远处来，不也快乐吗？人家不了解我，我却不怨恨，不也是君子吗？

```

说 —— 音读和意义跟“悦”字相同，高兴、愉快

愠 —— yùn，怨恨
君子 —— 《论语》的“君子”，有时指“有德者”，有时指“有位者”，这里是指“有德者”
```

> **子曰：道千乘之国，敬事而信，节用而爱人，使民以时**
> 
> 孔子说：治理具有一千辆兵车的国家，就要严肃认真地对待工作，信实无欺，节约费用，爱护官吏，役使老百姓要在农闲时间

```
乘 —— shèng，古代用四匹马拉着的兵车。春秋时代，打仗用车子，所以国家的强弱都用车辆的数目来计算，一千辆兵车算是大国
敬事 —— “敬”字一般用于表示工作态度，因之常和“事”字连用
爱人 —— 古代“人”字有广狭两义。广义的“人”指一切人群，狭义的人只指士大夫以上各阶层的人
使民以时 —— 古代以农业为主
```

> **子曰：弟子入则孝，出则悌，谨而信，泛爱众，而亲仁。行有余力，则以学文**
> 
> 孔子说：后生小子，在父母跟前，就孝顺父母；离开自己房子，便敬爱兄长；寡言少语，说则诚实可信，博爱大众，亲近有仁德的人。这样躬行实践之后，有剩余力量，就再去学习文献

```
入、出 —— “入”是“入父宫”，“出”是“出己宫”
谨 —— 寡言叫做谨
```

> **子夏曰：贤贤易色；事父母，能竭其力；事君，能致其身；与朋友交，言而有信。虽曰未学，吾必谓之学矣**
> 
> 子夏说：对妻子，重品德，不重容貌；侍奉爹娘，能尽心竭力；服事君上，能豁出生命；同朋友交往，说话诚实守信。这种人，虽说没学习过，我一定说他已经学习过了

```
子夏 —— 孔子学生，姓卜，名商，字子夏，比孔子小四十四岁
贤贤易色 —— “易色”解为“不重容貌”
```

> **子曰：君子不重，则不威；学则不固。主忠信。无友不如己者。过，则勿惮改**
> 
> 孔子说：君子，如果不庄重，就没有威严；即使读书，所学的也不会巩固。要以忠和信两种道德为主。不要跟不如自己的人交朋友。有了过错，就不要怕改正

```
重 —— 对自己有要求，言行不轻浮，举止有分寸，不随随便便
学则不固 —— 不庄重的人，学了也记不住，因为心浮气躁，学什么都是浅尝辄止
无友不如己者 —— 古今人对这一句发生不少怀疑，因而有一些不同的解释。但 AI 认为是不要和与自己志向不同、道德不同的人深交
惮 —— dàn，害怕
```

> **子禽问于子贡曰：夫子至于是邦也，必闻其政，求之与？抑与之与？子贡曰：夫子温、良、恭、俭、让以得之。夫子之求之也，其诸异乎人之求之与？**
> 
> 子禽向子贡问道：他老人家一到哪个国家，必然听得到那个国家的政事，求来的呢？还是别人自动告诉他的呢？子贡道：他老人家是靠温和、善良、严肃、节俭、谦逊来取得的。他老人家获得的方法，和别人获得的方法，不相同吧？

```
子禽 —— 陈亢，字子禽
子贡 —— 孔子学生，姓端木，名赐，字子贡，衞人，比孔子小三十一岁
夫子 —— 这是古代的一种敬称，凡是做过大夫的人，都可以取得这一敬称
```

> **子曰：父在，观其志；父没，观其行；三年无改于父之道，可谓孝矣**
> 
> 孔子说：当他父亲活着，要观察他的志向［因为他无权独立行动］；他父亲死了，要考察他的行为；若是他对他父亲的合理部分，长期地不加改变，可以说做到孝了

```
其 —— 指儿子，不是指父亲
行 —— xìng
三年 —— 古人这种数字，有时不要看得太机械。它经常只表示一种很长的期间
道 —— 有时候是一般意义的名词，无论好坏、善恶都可以叫做道。但更多时候是积极意义的名词，表示善的好的东西
```

> **有子曰：“礼之用，和为贵。先王之道，斯为美；小大由之。有所不行，知和而和，不以礼节之，亦不可行也”**
> 
> 有子说：“礼的作用，以遇事都做得恰当为可贵。过去圣明君王的治理国家，可宝贵的地方就在这里；他们小事大事都做得恰当。但是，如有行不通的地方，便为恰当而求恰当，不用一定的规矩制度来加以节制，也是不可行的”

```
礼与和是要达到平衡的 —— 制定规矩不是用来把人捆死的，不是机械性的守规矩，而是恰到好处，不刻板但也不乱来

案例：公司开会讨论一个错误
- 只有“礼”，没有“和”： 领导按章办事，当众严厉羞辱员工，虽然规矩守住了，但大家心冷了
- 只有“和”，没有“礼”（知和而和）： 领导怕得罪人，明明员工犯了错也不说，大家一团和气开完会，规矩废了，以后大家都会继续犯错
- “礼之用，和为贵”： 领导按照公司制度批评了错误，但私下沟通、就事论事，既解决了问题，又让员工心服口服
```

> **有子曰：“信近于义，言可复也。恭近于礼，远耻辱也。因不失其亲，亦可宗也”**
> 
> 有子说：“所守的约言符合义，说的话就能兑现。态度容貌的庄矜合于礼，就不致遭受侮辱。依靠关系深的人，也就可靠了”

```
不要什么事情都答应，我们只答应合乎道义的。
对人恭敬要符合礼节，不能过分恭敬，过分恭敬就成了谄媚了。
你所依靠的人如果是一个人品可靠的人，那么你就可以依靠他。例如职场中，你选择跟随一个领导。如果这个领导是一个人品正直的人（不失其亲），那么你的选择就是对的，是可以长久跟随的。如果你为了利益去投奔一个你内心深处并不认同、甚至人品低劣的人，最后多半会被过河拆桥

复 —— 兑现
远 —— 去声，音院，yuàn，动词，使动用法，使之远离的意思。此处亦可以译为避免
因 —— 依靠，凭借。有人读为“姻”字，那“因不失其亲”便当译为“所与婚姻的人都是可亲的”，恐未必如此。⑷宗——主，可靠。一般解释为“尊敬”，不妥。
```

> **子曰：“君子食无求饱，居无求安，敏于事而慎于言，就有道而正焉，可谓好学也已”**
> 
> 孔子说：“君子，吃食不要求饱足，居住不要求舒适，对工作勤劳敏捷，说话却谨慎，到有道的人那里去匡正自己，这样，可以说是好学了”

```
正 —— 匡正
```

> **子贡曰：“贫而无谄，富而无骄，何如？”。子曰：“可也；未若贫而乐，富而好礼者也”。子贡曰：“《诗》云：‘如切如磋，如琢如磨’，其斯之谓与？”。子曰：“赐也，始可与言诗已矣，告诸往而知来者”**
> 
> 子贡说：“贫穷却不巴结奉承，有钱却不骄傲自大，怎么样？”。孔子说：“可以了；但是还不如虽贫穷却乐于道，纵有钱却谦虚好礼哩”。子贡说：“《诗经》上说：‘要像对待骨、角、象牙、玉石一样，先开料，再糙锉，细刻，然后磨光’，那就是这样的意思吧？”。孔子道：“赐呀，现在可以同你讨论《诗经》了，告诉你一件，你能有所发挥，举一反三了”

```
子贡是商人出身说这句话可能也有让孔子表扬的意思。孔子是觉得这样做还不够，孔子觉得不要满足于"克制自己不犯错"，而要追求"根本不想犯错"
- 子贡描述的状态是克制，是不做坏事 —— 你在压制自己的某种冲动，穷了忍住不去谄媚，富了忍住不去骄傲，这需要刻意努力
- 孔子说的是忘我，是根本没想到去做坏事 —— 你根本就不需要克制，因为你的心思完全在别的地方。穷人沉迷于学问和道义，没空思考谄媚之事；富人则研究修养，不屑于是否骄傲

子贡又说了这就像玉石一样，每一道工序做完了，还有下一道，永远没有够了的时候，永远在精进。就像是人的修养，永远有更高的境界，永远不能觉得我已经够好了

赐 —— 子贡名。孔子对学生都称名
往 —— 过去的事，这里譬为已知的事
来者 —— 未来的事，这里譬为未知的事
```

> **子曰：“不患人之不己知，患不知人也”**
> 
> 孔子说：“别人不了解我，我不急；我急的是自己不了解别人”

```
患人之不己知 —— 为什么老板不发现我的才华
患不知人也 —— 我真的了解老板的需求吗？我了解公司的痛点吗？我了解同事们的优缺点以便更好地配合吗？

患人之不己知 —— 我这么优秀，他们怎么不爱跟我玩
患不知人也 —— 他们平时都在想什么？他们的为人准则是什么？我是不是误解了他们？
```

## 📖 为政篇
> **子曰：“为政以德，譬如北辰居其所而众星共之”**
> 
> 孔子说：“用道德来治理国政，自己便会像北极星一般，在一定的位置上，别的星辰都环绕着它”

```
孔子认为，治理国家不是靠“折腾”和“高压”，而是靠统治者自身的人格魅力，如果他本人技术过硬、为人正直、处事公平，他不需要天天盯着员工，大家也会自然而然地团结在他周围，朝一个目标努力
```

> **子曰：“道之以政，齐之以刑，民免而无耻；道之以德，齐之以礼，有耻且格”**
> 
> 孔子说：“用政法来诱导他们，使用刑罚来整顿他们，人民只是暂时地免于罪过，却没有廉耻之心。如果用道德来诱导他们，使用礼教来整顿他们，人民不但有廉耻之心，而且人心归服”

> **子曰：“吾十有五而志于学，三十而立，四十而不惑，五十而知天命，六十而耳顺，七十而从心所欲，不逾矩”**
> 
> 孔子说：“我十五岁，有志于学问；三十岁，懂礼仪、说话做事都有把握；四十岁，掌握了各种知识，不致迷惑；五十岁，得知天命；六十岁，一听别人言语，便可以分别真假，判明是非；到了七十岁，便随心所欲，任何念头不越出规矩”

```
- 15 岁时觉醒了，不再是父母老师逼着学，而是自己发自内心地想通过学习改变命运、理解世界
- 30 岁时已经形成了独立的思想，能自立于社会，不是指的买车买房成家立业，而是不在人云亦云，有自己的思想和价值观，独立了
- 40 岁见过各种人，别人怎么忽悠你，你心里都有数。你明白了自己的局限性，不再纠结，不再怀疑人生
- 50 岁认清了自己的使命，知道努力固然重要，但是有些事是自己无法控制的，而是尽自己所能，不再强求结果
- 60 岁听到什么话都能心平气和，不生气，你知道别人为什么骂你，为什么夸你，你不再因为外界的评价而情绪波动
- 70 岁已经把道德和规矩刻进了骨子里，随心所欲做出来的事情都是恰好符合规矩的
```
