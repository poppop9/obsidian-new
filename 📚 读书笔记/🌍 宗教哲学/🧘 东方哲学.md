# 易经
易经不属于任何一家，它是诸子百家的共同源头，它比儒、道、法、佛都更早，是中国思想的底层操作系统

易经教你现在是什么阶段、该进还是该退、该刚还是该柔

🔴 书单
- 曾仕强《易经的奥秘》 —— 入门最友好
- 南怀瑾《易经杂说》 —— 通俗有趣
- 傅佩荣《解读易经》 —— 学术但易懂
- 《周易》原文 + 《十翼》 —— 硬核选手


# 诸子百家
所有的诸子百家没有谁是对的，只有谁是合适的，人生不是选择题，是调色板，你要学会混搭，才能画出最适合你的颜色

读书不是让你变成无欲无求的圣人，  是让你变成一个要得起，也输得起的高手

| 时间    | 场景      | 用哪家       | 具体操作                      |
| ----- | ------- | --------- | ------------------------- |
| 早上8点  | 上班挤地铁   | 🟦 **道家** | "算了，挤就挤吧，反气也没用"（顺其自然，不内耗） |
| 上午10点 | 老板开会骂你  | 🟥 **儒家** | 表面恭敬，"好的领导我改进"（礼节，维持关系）   |
| 上午10点 | 内心OS    | 🟦 **道家** | "这傻X，我尽力就行"（心里不当回事）       |
| 下午3点  | 做项目遇到难题 | 🟥 **儒家** | 认真学习、请教同事、加班攻克（努力、进取）     |
| 下午5点  | 项目失败了   | 🟦 **道家** | "成事在天，我已经尽力了"（接受结果，不自责）   |
| 晚上7点  | 失恋了     | 🟪 **佛家** | "缘起缘灭，执着无用"（放下）           |
| 晚上9点  | 谈判加薪    | 🟨 **法家** | 准备筹码，列出成绩，冷静谈条件（利益博弈）     |
| 晚上11点 | 躺床上刷手机  | 🟦 **道家** | "去他的，明天再说"（放松）            |

幸福 ：外儒 + 内道 + 偶尔佛法
- 儒家 —— 对外人，讲礼貌、讲诚信、努力工作、赚钱养家
- 道家 —— 对内心，告诉自己工作就是个游戏、失败了也没关系，天道循环


## 儒家 —— 进
当你**迷茫、没动力**时，你应该学习儒家，它能给你**搞钱升职、成就感、归属感、家庭温暖**

如果你想在这个世俗世界里赢，信儒家

🔴 书单 ：《论语》、《孟子》、《资治通鉴》

## 法家 —— 进
大量分析权谋、制度、赏罚，教你怎么驾驭人心、控制局面，非常冷酷理性

法家是帝王的工具箱


🔴 书单
- 《韩非子》

## 道家 —— 退
当你**焦虑、压力大**时，你应该学习道家，它能给你**自由感、松弛感、不内耗**

容易真躺平，没钱吃饭，被人欺负

如果你想在这个内卷世界里保命，信道家

🔴 书单 ：《道德经》、《庄子》

## 佛家 —— 放
当你**痛苦、纠结、放不下**时，你应该学习佛家，它能给你**平静感、解脱感**

你痛苦是因为你执着。你想要钱、要爱、要长寿，但这些都会消失。只要你不执着，就没人能伤害你

太虚无了，容易看破红尘出家

失恋了、亲人离世、得了大病

🔴 书单 ：《心经》、《金刚经》

22 岁，精力最旺、试错成本最低、最需要**往前冲**的年纪，先别急着"看淡一切"，你还没拥有过，谈什么放下。应该 **7 分儒法（进攻） + 2 分道家（防崩） + 1 分佛家（备用）**

读完这三本，你会有一个清晰的人生坐标系：知道自己该往哪走，知道现在是"潜龙勿用"还是该"见龙在田"
1. 《论语》	建立做人的底层框架 —— 什么该做、什么不该做、怎么跟人相处。别读注释本，推荐杨伯峻《论语译注》，干净利落
2. 《孟子》	给你血性和志气。孟子是儒家里最刚的人，22岁读孟子会热血沸腾。推荐杨伯峻《孟子译注》
3. 曾仕强《易经的奥秘》	入门易经，理解"时位"概念——什么阶段做什么事，别在该拼的年纪躺平

读完这三本，你会变得不天真。你会明白：善良要带锋芒，信任要有边界，做事要讲策略
1. 《韩非子》	理解人性的阴暗面。不是让你变坏，是让你不被坏人欺负。重点读《说难》《五蠹》《孤愤》
2. 《资治通鉴》	历史就是人性的重复。推荐先读熊逸《资治通鉴熊逸版》降低门槛，或者柏杨白话版
3. 南怀瑾《易经杂说》	加深对易经的理解，学会判断形势、进退时机

读完这两本，你会获得一个情绪防弹衣：事情搞砸了不至于崩溃，被人骂了不至于内耗三天
1. 《道德经》，每章都很短，但每句都是炸弹。 推荐陈鼓应《老子今注今译》。教你：不争才是最高的争，示弱才是最大的强
2. 《庄子》	读完会觉得天地突然变大了，你那些焦虑都是笑话。推荐先读内七篇，用陈鼓应的注译本

佛学
3. 《心经》	260个字，遇到重大打击时读——失恋、亲人离世、重大失败。它会像一颗镇定剂
4. 《金刚经》	当你发现自己对某样东西执念太深、痛苦到无法自拔时读

三易 ：《归藏》《连山》《周易》，不是对同一道题给出了三个互相矛盾的答案，更准确地说，它们是同一个大系统的三个侧面/切入点
- 你从山顶看风景（连山）
- 你从地面看风景（归藏）
- 你从天空看风景（周易）

看到的景象不同，但描述的是同一个世界


万物从山中涌出，万物最终归藏于地，万物就这样运行不息
