# 📚 Non-Neediness
**Non-Neediness 的本质是你的自我价值和情绪稳定不依赖于他人的反应，你是给予者，而不是索取者**

> 你的快乐和自我价值是一个杯子里的水，当一个杯子是满的人遇到一个他欣赏的人时，他想做的不是求水，而是分享
> -  Neediness 的人 —— 杯子是空的。他迫切地需要别人往他的杯子里倒水（认可、喜欢、关注），否则他就会感觉自己一文不值，他的所有行为，都是为了求水喝
> - Non-Neediness 的人 —— 杯子是满的。他通过自己的生活（事业、爱好、朋友）把杯子装满了，他不需要别人来给他倒水，因为他自己就有水

🔴 欲望 vs. 需求 —— Non-Neediness 不是无欲无求
- Want 无需求感 = 我有欲望 + 我采取行动 + 希望有结果 + 不依赖结果
- Need 有需求感 = 我有欲望 + 我采取行动/不敢行动 + 希望有结果 + 极度依赖结果

🔴 **表现形式**
- 关于关注 ：一个专注于自己的事、不乞求别人关注的男人，反而显得神秘高价值

**Vulnerability 其实是 Non-Neediness 的行动证明 —— 只有当你真的 Non-Neediness，你才敢于冒着被拒绝的风险去展示真实的自己，这是一种自信和高地位行为**
