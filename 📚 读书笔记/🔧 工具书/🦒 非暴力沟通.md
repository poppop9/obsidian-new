
> [!NOTE]
> **非暴力沟通使用总纲**
> 
> 非暴力沟通是一把手术刀，只在做手术时拿出来（处理冲突、深层误解、情绪崩溃），日常请用普通工具（日常闲聊、吐槽、开玩笑）

> [!NOTE]
> **何时同理倾听，何时诚实表达**
> 
> 当对方遇到问题时，先通过同理倾听充分理解并回应对方。在对方被理解、情绪稳定之后，我们再诚实表达我们的困扰或需求

# 👂 同理倾听
**如果对方还处于某种情绪中，就很难与我们产生连结，这时我们要做的是同理倾听**

> **同理倾听成功的标志** —— 对方被充分同理后，内心就会释放更多的空间去关注其他的事情，我们能从他的身上观察到相应的变化
> ```
> 🌰 对方感到轻松，可能会深呼一口气
> 
> 🌰 对方安静下来停止说话
> 
> 🌰 对方开始愿意聆听你的解释
> ```

▶ **不打断对方讲话，并且允许对方打断自己讲话，因为你愈给对方机会发言，对方愈喜欢你**

```
🌰 面试时尽量让面试官多说
```

▶ **刻意停顿** —— 对方说完话时，让脸部表情呈现出正在消化内容的样子，过两秒左右，你再开口回应

▶ **完全卸下对他人先入为主的成见和评判** —— 你需要做的是不表达认同也不表示反对，更不把对方的话语当作攻击。因为一旦有评判，你听到的所有信息都会被用来验证你的评判，而不是去理解对方的真实感受

▶ **聆听每个信息（语言 + 非语言）中所隐含的需要** —— 人的所有情绪和行为背后，都源于某些未被满足的需要

▶ **用心和身体去感受对方，并用另一种方式协调回应** —— 对方紧张时你的心率也加快，对方难过时你的呼吸也略微沉重

```
🌰 如果你只模仿宝宝，仅仅说明你知道他的行为，而不是他的感觉。要让宝宝知道你能体会他的感受，你必须用另一种方式回应宝宝的内在感受，这时宝宝就会知道他得到了理解

🌰 在成年人的生活中，与母婴之间亲密协调最为接近的是做爱 —— 共同的欲望，一致的意图，同步转变唤起的相互状态，做爱的至高境界是具有同理心的情绪相互行为
```

▶ **倒映情绪，而非淹没** —— 跟随对方的频率，在心中映出对方的倒影，而不是被对方的情绪卷走

```
🌰 对方焦虑时，我保持平稳的语速、用温和的语气说“不急，你慢慢说，我听着”
```

# 🗣️ 诚实表达
**在对方尚未明白我们已经同理了他们的需要前，我们要避免提及我们自己的需要，因为这时的他们并不会在乎我们的需要。直到对方感到被充分理解后，他自然会开始愿意倾听你的解释，这时我们要做到诚实表达**

> 诚实表达的重点是用心，最好是将这项技能内化，而不是套流程

> 如果你正处于痛苦之中，无法同理别人，你可以表达出来自己当下的感受，或许别人会同理你

▶ **诚实表达的禁忌**
- 给建议 ：我觉得你应该 ……
- 比惨 ：这算什么，我 ……
- 安慰 ：这不是你的错
- 讲故事 ：这让我想到 ……
- 询问 ：持续多久了 ……

▶ **诚实表达四要素**
1. **观察** —— 不夹杂评判的观察实际上发生了什么，不论喜欢与否，我们只是说出人们做了什么
2. **感受** —— 表达出我们看到这些行为时的感受（伤心、害怕、喜悦、有趣，心烦），注意只是你自己的感受，而不是对他人的想法
3. **需要** —— 表达出我们的需要
4. **请求** —— 一个清晰具体的请求（希望对方做什么，不要做什么），注意是请求，而非要求

```
🌰 我想要吃意面，但是被妈妈反对
- 妈，你是不是有点担心我总吃意面什么的，营养不均衡、对身体不好？是不是你一直希望我吃得更有营养、身体更健康？（如果对方已经感觉到被同理了，你就可以诚实表达了）
- 听你这么说，我心里有点难过，因为我其实不是刻意要特立独行，而是真的今天特别想吃点轻松点的东西，你看我就煮一包，不多，保证我也会注意营养，下顿按你说的吃，好吗

🌰 朋友借东西但从不归还。你可以说 ：老赵，看到你想借耳机的消息，我其实有点不好受……，因为我突然想到笔记本上次借你到现在还没还，我最近其实也要用。我知道你不是故意的，只是我也希望咱们借东西的时候可以更及时归还，这样下次我也能安心再借你。你觉得呢

🌰 我看到咖啡桌下的两团脏袜子和电视机旁边的三团脏袜子，我很生气，因为我希望在我们共用的空间里能多些整洁，你愿意把你的袜子放在你房间或放进洗衣机里吗

🌰 你说严肃的事情，被大家嘲笑文艺伤感青年。你可以说 ：我知道大家可能觉得我说得有点沉重，但这是我最近真的在经历的事，我在这里不是想当气氛终结者，我说出来可能会好受一点，也是真的想被理解一下，能多谢你们听我说完
```

# ☮️ 化解冲突
**化解冲突的关键在于建立双方的联结，只有当彼此真正联结，才会愿意从心底理解对方的感受与需求，而许多问题也会随之迎刃而解**

> **错误方式** —— 双方总在证明自己是对的、对方是错的，而不关注自己和对方的需要

> **错误语言** —— 可是、但是。这些词会引起对方不适

▶ **如何化解冲突**
1. 首先，让双方表达自己的需要 —— 如果他们表达的是想法、评判、分析，我们可以继续寻找话语背后的需要，直到双方都准确理解了彼此的需要
2. 其次，清楚需要后要尽可能地同理双方，以便让双方都能确切地了解彼此的需要
3. 然后，我们就可以用正向的、描述行动的、此刻可回应的语言提出化解冲突的策略，从而持续的让双方进行连结 —— 如果表达的语言不是此刻可回应的，对方便无从知晓当下可以怎么做
4. 最后，如果双方就解决策略依旧存在问题，那你就要做出选择了，到底是要妥协哪一方

```
🌰 伴侣因为回家不收拾起冲突，A 说 ：你每次回家都把东西乱丢，家里像旅馆一样。B 说 ：我上班已经很累了，回家还要被你念
- 同理 A —— 你看到东西堆着会很烦躁，是因为你希望家里是有秩序的、整洁的，对吗？所以你希望有人一起分担，而不是都压在你身上，是吗？
- 同理 B —— 你下班回家已经很累了，如果一进门就被指责，你会觉得很压迫，你其实需要休息和被理解，对吗？所以你不是拒绝整理，你是需要先有个缓冲时间再来做，对吗？
- 提策略 —— 我听见你很需要秩序，我也听见你需要下班后的缓冲，我们来看看，有没有一个方式可以同时照顾这两个需要，B 能不能每次回家先休息 20 分钟，然后把包、外套、杯子放回固定位置，如果你今天真的太累，能不能直接告诉我需要先休息半小时，半小时后来收
```

# 🫶 感激
## 📖 表达感激
**表达感激的唯一意图就是庆祝他人的作为是如何使我们的生命变得更加丰盛的，而不是索取回报**

▶ **具体方向** —— 他人做了什么增进了我们的福祉、我们有哪些需要因此得到了满足、因为这些需要得到了满足，引起了我们什么样的愉悦感受

```
🌰 感谢你，我一直无法和我的儿子沟通，并一直在寻找一种情意相通的方式和他相处，你说的这两点让我找到了方向
```

## 📖 接受感激
**把他人的感激当作是给自己的礼物，想想自己帮到了别人什么，说说自己此刻的感受**

> 别说 “没什么大不了的”，这样会让被人忽略我们的帮助

```
🌰 能为你解决这个难题，我也感到很轻松

🌰 如果我们的立场互换，我相信你也会这么做
```
