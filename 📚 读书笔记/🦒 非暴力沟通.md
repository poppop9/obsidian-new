# 📚 互相给予
🏷️ 只要我们发自内心地给予，并尽我们所能让对方知道这是我们唯一的动机，对方也会加入我们，最终双方必然能以善意相待

🏷️ 让双方都能发自内心地相互给予，我们需将注意力聚焦在四个方面
1. 首先，**观察** ：我们要不夹杂任何评判的观察实际上发生了什么，不论喜欢与否，我们只是说出人们做了什么
2. 第二步，**感受** ：表达出我们看到这些行为时的感受（伤心、害怕、喜悦、有趣，心烦），注意只是你自己的感受，而不是对他人的想法
3. 第三步，**需要** ：表达出我们的需要
4. 第四步，**请求** ：一个具体的请求

🏷️ 例如，一位母亲和儿子用非暴力沟通的方式来说话 ：“我看到咖啡桌下的两团脏袜子和电视机旁边的三团脏袜子，我很生气，因为我希望在我们共用的空间里能多些整洁，你愿意把你的袜子放在你房间或放进洗衣机里吗？”



# 📚 聆听
持续地聆听每一句话背后所包含的感受和需要，不表达认同或不认同，也不将他的话当作攻击。这样慢慢的他人就会分享他灵魂深处的脆弱，当他人感受到被充分理解后，便开始愿意听你的解释
