# 📚 调式
**调式是可移调的、没有主音概念的音程模式**

西方调式体系演化 ：中古教会调式 → 大小调体系 → 近现代调式看，后者总是在前者的基础上演化

## 📖 教会调式
**起源于中世纪欧洲的教会音乐（约 9–16 世纪），是一套古老的音阶系统**
- 音阶结构固定，每种调式由特定的全音/半音排列决定
- 旋律导向：注重横向旋律线条的自由流动
- 弱功能和声：没有属到主的强烈解决倾向，缺乏大小调那种明确的回家感
- 各调式之间的色彩差异主要来自半音位置的不同

教会调式有七种调式 —— 可以理解成从白键的 C、D、E、F、G、A、B 依次开始，弹出一个八度的白键音阶，每个起始音产生不同的全半音排列，就是不同的调式

| 调式名称                  | 音程结构          | 对应白键起始音 | 色彩印象        |
| --------------------- | ------------- | ------- | ----------- |
| **伊奥尼亚（Ionian）**      | W-W-H-W-W-W-H | C       | 明亮、愉快       |
| **多利亚（Dorian）**       | W-H-W-W-W-H-W | D       | 柔和的小调、略带明亮  |
| **弗里吉亚（Phrygian）**    | H-W-W-W-H-W-W | E       | 阴暗、异域、西班牙风情 |
| **利底亚（Lydian）**       | W-W-W-H-W-W-H | F       | 梦幻、飘浮、开阔    |
| **混合利底亚（Mixolydian）** | W-W-H-W-W-H-W | G       | 明亮但带蓝调感、摇滚  |
| **爱奥利亚（Aeolian）**     | W-H-W-W-H-W-W | A       | 悲伤、柔和       |
| **洛克利亚（Locrian）**     | H-W-W-H-W-W-W | B       | 不稳定、紧张      |

## 📖 大小调体系
从教会调式中演化而来，17 世纪前后在欧洲确立，巴洛克→古典→浪漫时期达到巅峰（约 1600–1900 年），本质上是从教会调式中筛选出了两种最具功能性的调式 Ionian、Aeolian
- 功能和声 ：和弦被赋予主 T — 下属 S — 属 D的功能角色
- 强主音中心 ：音乐有明确的家（主和弦），离开后渴望回归

> C 大调 ➜ 把起始音移到 C 上的 Ionian 调式
> D 大调 ➜ 把起始音移到 D 上的 Ionian 调式
> C 小调 ➜ 把起始音移到 C 上的 Aeolian 调式

### 📝 大调体系

| 类型                               | 音程结构          | 说明       |
| -------------------------------- | ------------- | -------- |
| **自然大调（Natural Major / Ionian）** | W-W-H-W-W-W-H | 明亮、稳定、积极 |

### 📝 小调体系

| 类型                                | 音程结构            | 与自然小调的区别  | 色彩/用途             |
| --------------------------------- | --------------- | --------- | ----------------- |
| **自然小调（Natural Minor / Aeolian）** | W-H-W-W-H-W-W   | —         | 悲伤、柔和、暗淡          |
| **和声小调（Harmonic Minor）**          | W-H-W-W-H-WH-H  | 升高第 7 级   | 中东风情、古典戏剧感        |
| **旋律小调（Melodic Minor）**           | ↑ W-H-W-W-W-W-H | 升高第 6、7 级 | 旋律更平滑流畅，消除增二度的突兀感 |

关于和声小调
- **为什么要升第 7 级？** 自然小调的第 7 级（♭7）距主音是全音，缺乏"导音→主音"的半音解决（不够紧张）。升高第 7 级后，产生了导音，使 V→i 的解决力大大增强
- **副作用：** 第 6 级和升第 7 级之间出现了**增二度（Augmented 2nd）**音程，听感上有一种独特的"中东/阿拉伯风味"，但在旋律线中被认为不够平滑

关于旋律小调
- **上行**：同时升高第 6 级和第 7 级 → 消除增二度，旋律线平滑上行
- **下行（传统规则）**：还原第 6、7 级 → 等于自然小调下行，听感更柔和自然
- **现代用法（爵士旋律小调）**：上下行统一使用升 6、升 7，不再区分方向。这在爵士乐理中非常重要，是许多爵士调式的母音阶

## 📖 近现代调式
19 世纪末至 20 世纪，作曲家们对大小调体系感到审美疲劳，开始寻找新的音响色彩，主要方向是向前探索全新的音列组织方式、向后复兴古老的教会调式
- 打破或模糊传统的主音中心
- 追求新的音响色彩和音阶结构
- 强调音色和色彩而非功能和声


# 📚 移调
**移调是在不改变音程模式的前提下，把整个调式平移到不同的音高**
- 移调会改变调式的调式主音（音乐重心），但音乐的色彩和情绪特征保持一致

# 📚 音阶
**音阶是在将某一调式移调至特定起始音后，通过选定一组特定音而形成的具体音高序列**
- 同一个调式可以通过移调、选定特定音产生无数个不同的音阶

## 📖 五声音阶
🔴 大调五声音阶 / 民间五声音阶 / 中国五声音阶 ：以 Ionian 调式的音程模式为基础，起始音定为 C，然后去掉原音阶中的 F 和 B，得到 C D E G A
- 为什么去掉 F 和 B ：因为这两个音会产生半音、紧张感
- 平和、自然

# 📚 调性
调性是在调式的基础上，以某个音为中心的音高组织系统，具有明确的稳定音和不稳定音之间的倾向关系（调性 = 调式 + 和声功能  + 解决倾向）
