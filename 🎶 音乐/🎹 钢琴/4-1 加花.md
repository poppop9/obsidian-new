
# ❤️ 调整节奏
调整节奏加的拍子一般是和弦内音

>[!quote] 和弦内音
>和弦内音 就是在和弦里面的音
>- 和弦内音加进来是不会影响听感的
>- 只要弹和弦内的音，节奏正确，随意变化，都可以

---

- **一拍** : （x x x x） 平稳感
- **两拍** : 本来是 1s 一拍（x x x x） ➡️➡️➡️ 我现在可以 1s 两拍（x x x xx），一般加的那一拍是和弦内音，推动感
	- 比如一级和弦，可以加 .5
- **三拍** ：x x x xxx
- **四拍** : 1s 一拍 --->>> 1s 四拍（x x x xxxx），会有一种爆发感，适合用在进副歌
	- 比如一级和弦，可以加 5 3 1 .5 或者 .5 1 3 5

# ❤️ 长拍加花
- 四四拍中，节奏大于 5 拍的同和弦长拍加花
```
3         3
7 - - - | 7 - - 0 |

        ⬇

3            3
7 - 03 #57 | 3. - - 0 |
```
- 四四拍中， 前后同和弦的 1 个小节的长拍加花
```
1         1
1 - - - | 0 5 - - |

        ⬇

15 1.2. 3.2. 3.5. | 1..
```
- 四四拍中，前后不同和弦的 1 个小节的长拍加花
```
1         4
1 - - - | 0 5 - - |

        ⬇

15 1.2. 3.5. 1..2.. | 1..
```

```
1         4   5
1 - - - | 1 - - - |

        ⬇
                            5..
                      6.    2..
15 1.2. 3.5. 1..5.. | 4.. - 7. - |
```
