@Entity

     * 初始化数据

我希望自定义一个注解，用于 activityOrderId 属性上，可以自动生成     @PrePersist
public void setCreatedAt() {
	if (this.activityOrderId == null) {
		this.activityOrderId = IdUtil.getSnowflakeNextId();
	}
}
这样我就不用在类里写这样一个方法了，我就直接加一个注解就好了，这个注解有一个属性，可以传入要如何生成，比如最后的效果：
@JpaDefaultValue(howToCreate = cn.hutool.core.util.IdUtil.getSnowflakeNextId())
private Long activityOrderId;  // 活动单id
