- `@Validated` 在类上加这个注解，校验参数的具体 message 才会返回
- `@Valid` 
- 参数校验
	- `@NotBlank` 

```java
@Validated  
@RestController  
@RequestMapping("/api/finance/lCLCustomers")  
public class FinanceLCLCustsController {

	@GetMapping("/queryList")
	public TaoResponseMsg queryList(@RequestParam @NotBlank(message = "不允许为空文本") String custName,
	) {
		TaoResponseMsg taoResponseMsg = new TaoResponseMsg();
		taoResponseMsg.addValue("data", financeLCLCustsService.queryReferenceListByLCLCust(custName, littleof_pagenum, littleof_size));
	
		return taoResponseMsg;
	}

}
```
