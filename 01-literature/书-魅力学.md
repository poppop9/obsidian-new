# 📚 魅力的四种常见风格

## 📖 专注型魅力
**专注型魅力就是「临在感」+「尊重他人」，让对方喜欢他们自己，并觉得他们自己的意见很重要**

```
马斯克、比尔盖茨、前 IBM 总裁基勒

管理顾问、服务领域、律师、会计师、财务顾问
```

👉 **平衡**
- 需要一点亲和力 —— 没有亲和力会让人觉得咄咄逼人
- 需要一点影响力 —— 没有影响力会让人觉得你姿态很低，可能有求于他

👉 **适合场合** —— 商业场合、谈判、化解冲突、需要别人敞开心胸、分享信息时

👉 **不适合场合** —— 有必要展现权威、需要人们听从指示、不希望对方对你倾吐太多时

> *不必让他人觉得你多聪明，只需让他们觉得自己聪明就行了*

## 📖 仁慈型魅力
**仁慈型魅力主要来自于「亲和力」，使对方感到备受欢迎、被全心全意的接纳与包容**

```
达赖喇嘛
```

👉 **平衡** —— 需要一点影响力，没有影响力会让人觉得你想要取悦他人

👉 **适合场合** —— 宣布坏消息、应付难以相处的人时

👉 **不适合场合** —— 有必要展现权威、不希望对方对你倾吐太多时

## 📖 远见型魅力
**远见型魅力主要来自于「影响力」，能带给人们信念，让人感到热血沸腾**

```
乔布斯
```

👉 **如何展现远见型魅力** —— 必须向他人投射出对某个理想抱持坚不可摧的信心，并且必须涵盖某种程度的「高贵情操」与「利他行为」

👉 **平衡** —— 需要一点亲和力，没有亲和力会让人非常讨厌

👉 **适合场合** —— 激励人心的场合、危机时刻

> *我们天生讨厌不确定感，在变化多端的世界，我们渴望有个明确的东西能握在手中。这就是远见型魅力威力十足的原因*

## 📖 权威型魅力
**权威型魅力主要来自于「影响力」**

👉 **平衡** —— 需要一点亲和力，没有亲和力会让人非常讨厌

👉 **适合场合** —— 希望别人照你的指示做事时、危机时刻

👉 **不适合场合** —— 想要激发创造力（因为权威会抑制他人的批判性思考）、希望获得建设性回馈时

# 📚 魅力的阻碍

## 心理上的不适
**心理上的不适（自我怀疑、焦虑、比较）会削弱我们的临在感、会产生焦虑、还会产生负面情绪、让我们的肢体语言不自然**

👉 **解决办法一 ：责任转移** —— 当我们面对不确定时，其实最想要的是事情能顺利进行，每件事都有人负责，你可以进行以下练习
1.  舒服的坐着 / 躺下，放轻松，闭上眼睛
2.  做三次深呼吸 —— 吸气时，想像自己将纯净的空气引导到头顶；呼气时，让这股气息快速窜流全身，洗净所有的忧虑与烦恼
3.  选择某个仁慈的神祇 ── 上帝、命运之神、宇宙
4.  想像把肩上扛着的忧虑卸下，转移到你选定的神祇肩上，它能够替你托管心中所有的烦恼
5.  观想自己肩上的所有重责全数卸下，现在的你不必再对任何事物的结果负责，每件事都已获得托管

👉 **解决办法二 ：将不适感去污名化** —— 告诉自己“你不是世上唯一有这种内在不适的人，还有很多人和你一样正在经历你目前正在承受的痛苦”，把它想像成一群人在受煎熬，这个难熬的重担是大家在一起承担

> **案例** ：假设你刚失去了一个重要客户，不妨想想认识的人（某位精神导师、仰慕的人）当中是不是也经历过类似的挫折，想想他们是怎么熬过来的。当然，若是本人能亲口向你诉说他们的故事是最理想的

👉 **解决办法三 ：中和负面情绪**
- 不要以为自己的想法都是对的。对方可能只是因为内心不舒服才会显现出这样的神情。你感受到的可能只是对方心里极度不适而外显的震波，这个负面震波极可能和你毫无关系。而且我们的大脑倾向专注在危险的事物上，它认为重要的，往往是最负面的
- 把你的想法当成墙上的涂鸦、或是大脑表面电流撞击而生的细小火花
- 想像从远方观看自己。把镜头拉到天际，远到可以看到地球漂浮在太空。接着，拉近一点，看到你所处的大洲、你的国家、你的城市、最后是你待的房间。看看渺小的你，以及大脑里头电流撞击的飕飕响声──某个渺小的个体在此时此刻体验到的某个感受
- 想像最糟的情况，并了解无论情况多么糟糕，你都能够挺得过去

👉 **解决办法四 ：改写事实** —— 把事情往好的方面想，例如如果你焦虑演讲，你可以幻想自己成功的场面越细节越好，最好把过程写下来。例如你怨恨他人，但是又不得不与他相处，可以把他的道歉过程想象出来并写下来

👉 话题

# 优雅的退场
人们喜欢与富含魅力的谈话高手聊天，但这么一来，对方可能会因聊得太开心而不想放你走。

首先，对话宜尽早结束，别拖太久。不然你和对方都会感到压力而不自在。最简单的方式当然就是有个非走不可、冠冕堂皇的理由──或许是你担任宴会的志工，或是有公务要忙。别人一听到你「有任务在身」，自然就不会预期你能聊太久。

要是你必须打断某一组人的谈话该怎么办呢？或许你必须从某个谈话圈里，拯救出某位有意离开的人；或者你需要介绍其中的人给另外的人认识；又或者是其中一人为了职务必须离开。
碰到这种状况时，你必须集中注意力，带着特别友善的眼神注视对方（参见第九章的技巧）──不是看着你要带走的人，而是被留下来的那个人。这么做是尽量不让对方觉得自己受到排挤，此时尤其要用仁慈型或专注型的魅力来表达。这时候，最好的措辞方式应该像：「真的很抱歉，但克里斯多夫必须先去忙（某某事情）。你可以容许我带他离开吗？」这么说至少让对方感觉决定权在他们手里。

> 影响人们的不是你用的字眼或是说话的内容。事实上，人们只记得跟你讲话时的「感受」

# 沟通
会让人往坏处想的，不只是隐喻而已；某些我们常用的辞汇也有同样的效果。譬如当你告诉某人「没有问题」、「不要担心」、或是「不要犹豫打电话给我」时，他们可能只记住「有问题」、「要担心」、或是「要犹豫」的字眼，而不是你想要支持他们的心意。为了消除这类负面影响，不妨改成这么说：「我们会处理的」、或是「随时想打电话给我都行」

收到别人的赞美时，你的感觉如何？打个比方，当对方说你好看，或是你完成了某件事让他很感动，你的第一个反应是不是想淡化它？对多数人来说，听到赞美，会既高兴，又带点尴尬，不太知道该如何应对。我们多半不是突然害羞起来，就是谦虚的想转移话题：「哦！这没什么……。」
遗憾的是，这么做反而传达一个消息给赞美你的人：他们赞美你是不对的。他们可能因此觉得自己相当愚蠢，甚至产生联想，认为你让他们像个笨蛋。一旦你常让他们有这种感受，很快的他们就不想再赞美你了。不过，若是你让他们觉得恭维你是件好事，他们对自己的感觉也会变好，因而想继续赞美你。下次有人称赞你时，不妨采用以下步骤，巧妙应对：
•  停下来。
•  咀嚼对方的恭维。如果可以，好好享受它。
•  做出你正在消化赞美的表情。让对方感受到他的赞美对你起了作用。
•  感谢他们。说声「非常谢谢你」就可以了，你还能进一步感谢他们如此贴心，或告诉他们，他们的赞美会让你开心一整天
