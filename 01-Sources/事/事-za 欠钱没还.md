# 背景
我在厚街合唱团认识了一个朋友，关系不生不熟，认识了几个月，每周合唱团排练时会见面，由于住的很近，偶尔会一起走回去。

如果只听他讲述自己的经历，会觉得他挺真诚的
- 他说自己欠了很多钱，他并没有隐藏自己
- 他说他会花很多钱请朋友吃饭（~~但是朋友生日聚会不叫他，他觉得别人没把他当朋友~~）
- 他说了自己的创业失败经历，工作经历（~~真实性有待考究，我的直觉告诉我应该是真的~~）

但看他的行为，就觉得这人一般
- 借钱两次爽约
- 微信聊天懒得接住别人情绪，只有我接住他的情绪，他只会在对别人有需求时才接触别人（~~这点我跟他前女友的评价一样~~）
- 雷老师教了他这么久的合唱，最后表演填写报名表时，他的考虑因素居然只有他自己，没有考虑过整个合唱团的低声部人不够，需要他

> 综合以上几点，我觉得[判断一个人时，不要看他说什么，要看他做什么](../../02-Permanent/未完成/判断一个人时，不要看他说什么，要看他做什么.md)

# 经过
**06-10（借钱）** 一起走回去，路上他问"有没有人找你借过钱"，我忘记我回答了什么。到他家楼下，他提议坐下聊聊，聊的基本都是他的经历。中途又问了一次"如果我找你借钱你会怎么样"，我说“我觉得很正常，你找这么多人借过钱，如果找我借我觉得很正常”。到最后聊的差不多了，准备回去了，他叫住了我，说“你先坐下，我真的要找你借钱”，我说“可以啊，但是我能借你的不多”，他说“200”，我说“好，我转你”，他说“19 号还你，也就是节目演出那天”，最后我支付宝转了他 200。

**06-19（第一次爽约）** 节目排练当天，我们一起打车过去，刚见面就说"本来有钱还你的，但电子烟杆坏了，买完就没钱了"。改口说 25 号还，那天他有收款。

**06-19 ~ 06-25（中间）** 他有来找我微信聊天，但没提还钱的事。

**06-25（第二次爽约）** 微信又来找我聊天，没提还钱

# 反思
1. **我对他真的"没有需求"吗？** 每次微信聊天都是他找我，每次散步坐下聊也是他提议。表面上看，我对他没有需求。但他给我讲了很多他的经历，我是不是潜意识里觉得"他以后可能帮到我"，所以才愿意接他的情绪、借他钱？我自己也不确定
2. **预判了，但预判不能替代感受** 借钱时就已经料到他可能不还。因为他自己说还欠很多人的钱。我也做好了不还的心理准备。但当他真的没还时，还是不自在。说明"理性上接受"和"情绪上消化"是两件事
