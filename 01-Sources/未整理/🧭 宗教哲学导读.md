# 📚 宗教的演化史
## 泛灵论（采集时代）
**万物有灵（石头、树木、河流都有灵魂），人和动物是平等的，没有谁比谁高贵。地方特色较重，每个部落信不同的灵，无法统一大批人**

## 多神教（农业革命后）
**农业革命后，人需要和神灵做交易（我献祭给你，你保佑我丰收），所以出现了各种专职的神（丰收之神、战争之神、爱情之神 ……）**

▶ **包容** —— 多神教天然就接受别的神也存在，多神教很少因为信仰而迫害异教徒

## 一神教
**最初犹太教只是说"耶和华是我们的唯一神"，并没有说别人的神不存在，后来基督教和伊斯兰教把它升级为宇宙间只有一个真神，其他的都是假的**

> **案例** ：犹太教、基督教、伊斯兰教

▶ **排他性、传教冲动** —— 既然只有一个真神，那所有人都该信祂，不信的人就是异端

## 自然法则宗教
**自然法则宗教是一种没有神的宗教**

> **案例** ：佛教、道教、儒教、斯多葛学派、伊壁鸠鲁学派

### 现代的自然法则宗教
| 意识形态          | "自然法则"是什么                 | 说明                       |
| ------------- | ------------------------- | ------------------------ |
| **自由人文主义**    | 人的自由意志和个人权利是"自然的"、"不可剥夺的" | 其实"人权"不是自然界的客观法则，而是虚构的故事 |
| **社会主义/共产主义** | 历史发展规律（阶级斗争→共产主义社会是必然的）   | 马克思认为自己发现了"历史的自然规律"      |
| **资本主义**      | 市场这只"看不见的手"是自然法则          | 亚当·斯密认为自由市场会自然达到最优       |
| **进化人文主义**    | 自然选择/优胜劣汰是最高法则            | 纳粹认为自己在"遵循自然规律"淘汰劣等种族    |
| **生态主义/环保主义** | 生态平衡是最高法则                 | 人类应该服从自然生态系统的规律          |

# 🧘 东方宗教哲学
诸子百家没有谁是对的，只有谁是合适的，人生不是选择题，是调色板，你要学会混搭，才能画出最适合你的颜色

读书不是让你变成无欲无求的圣人，而是让你变成一个要得起，也输得起的高手

| 时间    | 场景      | 用哪家       | 具体操作                      |
| ----- | ------- | --------- | ------------------------- |
| 早上8点  | 上班挤地铁   | 🟦 **道家** | "算了，挤就挤吧，反气也没用"（顺其自然，不内耗） |
| 上午10点 | 老板开会骂你  | 🟥 **儒家** | 表面恭敬，"好的领导我改进"（礼节，维持关系）   |
| 上午10点 | 内心OS    | 🟦 **道家** | "这傻X，我尽力就行"（心里不当回事）       |
| 下午3点  | 做项目遇到难题 | 🟥 **儒家** | 认真学习、请教同事、加班攻克（努力、进取）     |
| 下午5点  | 项目失败了   | 🟦 **道家** | "成事在天，我已经尽力了"（接受结果，不自责）   |
| 晚上7点  | 失恋了     | 🟪 **佛家** | "缘起缘灭，执着无用"（放下）           |
| 晚上9点  | 谈判加薪    | 🟨 **法家** | 准备筹码，列出成绩，冷静谈条件（利益博弈）     |
| 晚上11点 | 躺床上刷手机  | 🟦 **道家** | "去他的，明天再说"（放松）            |

最高境界 —— 说儒家话 做道家人 行法家事 修佛家心
- **儒家（孔子）是粮店：** 是主食，天天都要吃，教我们如何做人做事，不吃就会饿死
- **道家（老庄）是药店：** 人生生病了，遇到过不去的坎儿了，或者想不通了，就要去道家那里抓点药，吃完病就好了
- **法家（韩非子）是五金店（或者是武器库）：** 里面全是锤子、钳子、刀枪，平时不能乱摆在明面上，但要解决实际困难、防身自卫、建立规矩时，非去里面拿工具不可

## 儒家 —— 进
[[../书/书-论语]]

当你**迷茫、没动力**时，你应该学习儒家，它能给你**搞钱升职、成就感、归属感、家庭温暖**

如果你想在这个世俗世界里赢，信儒家

▶ 书单 ：《论语》、《孟子》、《资治通鉴》

## 法家 —— 进
大量分析权谋、制度、赏罚，教你怎么驾驭人心、控制局面，非常冷酷理性

▶ 书单《韩非子》

## 道家 —— 退
[[../书/书-道德经]]

当你**焦虑、压力大**时，你应该学习道家，它能给你**自由感、松弛感、不内耗**

容易真躺平，没钱吃饭，被人欺负

如果你想在这个内卷世界里保命，信道家

▶ 书单 《道德经》、《庄子》

## 佛家 —— 放
当你**痛苦、纠结、放不下**时，你应该学习佛家，它能给你**平静感、解脱感**

你痛苦是因为你执着。你想要钱、要爱、要长寿，但这些都会消失。只要你不执着，就没人能伤害你

失恋了、亲人离世、得了大病

▶ 书单 ：《心经》、《金刚经》

## 学习顺序
22 岁，精力最旺、试错成本最低、最需要往前冲的年纪，先别急着"看淡一切"，你还没拥有过，谈什么放下。应该 **7 分儒法（进攻） + 2 分道家（防崩） + 1 分佛家（备用）**
1. 读完这三本，你会有一个清晰的人生坐标系：知道自己该往哪走，知道现在是"潜龙勿用"还是该"见龙在田"
	- 《论语》	建立做人的底层框架 —— 什么该做、什么不该做、怎么跟人相处。推荐杨伯峻《论语译注》
	- 《孟子》	给你血性和志气。孟子是儒家里最刚的人，22 岁读孟子会热血沸腾。推荐杨伯峻《孟子译注》
	- 曾仕强《易经的奥秘》入门易经，理解"时位"概念，什么阶段做什么事，别在该拼的年纪躺平
2. 读完这三本，你会变得不天真。你会明白善良要带锋芒，信任要有边界，做事要讲策略
	- 《韩非子》理解人性的阴暗面。不是让你变坏，是让你不被坏人欺负。重点读《说难》《五蠹》《孤愤》
	- 《资治通鉴》历史就是人性的重复。推荐先读熊逸《资治通鉴熊逸版》降低门槛 or 柏杨白话版
	- 南怀瑾《易经杂说》加深对易经的理解，学会判断形势、进退时机
3. 读完这两本，你会获得一个情绪防弹衣，事情搞砸不崩溃，被人骂了不至于内耗三天
	- 《道德经》 推荐陈鼓应《老子今注今译》。不争才是最高的争，示弱才是最大的强
	- 《庄子》	读完会觉得天地突然变大了，你那些焦虑都是笑话。推荐读陈鼓应的注译本
4. 佛学
	- 《心经》	遇到重大打击时读 —— 失恋、亲人离世、重大失败。它会像一颗镇定剂
	- 《金刚经》当你发现自己对某样东西执念太深、痛苦到无法自拔时读
