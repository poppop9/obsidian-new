# literature 结构
literature 即文献笔记
- 书籍记作 `书-书名`，若单个 md 太多了，可以细分为 `书-书名-章节号-章节名`
- 公众号文章、网页、抖音、AI 获取的知识记作 `文-文章主题`
- 人物介绍记作 `人-人名`

# 笔记符号
### 导航类
- `§`：节 / 区块 / MOC 分节
- `@`：人物 / 作者 / 机构 / 理论来源

### 思考状态类
- `!`：洞见 / 值得保留的结论
- `?`：问题 / 未决点 / 待验证
- `~`：暂定观点 / 草稿表达
- `⚠`：边界 / 例外 / 风险

### 关系类
- `→`：引出 / 支持 / 导致 / 推演
- `↔`：对照 / 互补 / 张力
- `↑`：上位抽象 / 原则化
- `↓`：下位实例 / 应用场景
- `×`：冲突 / 反驳 / 不兼容
