# 📚 笔记编写规范
```
- 一级标题：📚 (书本，表示整个学科/主题)
- 二级标题：📖 (打开的书，表示章节)
- 三级标题：📝 (备忘录，表示知识点)
- 四级标题：✏️ (铅笔，表示细节/要点)

全篇的补充说明（一般写在开头） > [!NOTE]
要点 👉，感觉要改成粗体开头加空格就好了
次级要点 在每个标题的总结下新建一个段落
案例 ``` 代码块
自己的想法（最好是写在后面） 引用 >

突出文字重点 ： 「」
```

## 📖 内部超链接

```

```

```
[[#三、超链接]]
```
- 链接章节详细内容
```
![[#^7b4f97]]
```
- 链接全局
```
![[^^存储 运算 控制]]
```

# 📚 快捷键
- 新建笔记 `ctrl+n`
- 搜索笔记 `ctrl+o`
- 在当前笔记中搜索 `ctrl+f`
- 管理工作区 `ctrl+w`
* 打开命令面板 `ctrl+p`
* 切换编辑/阅读视图 `alt+v`

* `shift + ←` 向左选中文字
* `shift + →` 向右选中文字

* 插入链接 `ctrl+k`
* 无序列表 `ctrl+l`
- 粗体 `ctrl+b`
- 斜体 `ctrl+i`
- 删除线 `ctrl+d`
- 单行代码块 ``ctrl+`一个``
- 清除格式 `ctrl+e`
- 引用 `ctrl+q`

- excalidraw 里拷贝内部链接 `ctrl+shift+k`

# 📚 插件
- `Better footnote` 阅读视图时，脚注可以鼠标悬浮显示
- `Commander` 控制在侧边栏，底部，各处的按钮
- `Editing Toolbar` 悬浮的快捷编辑栏
- `Excalidraw`
- `Image auto upload Plugin` 图床
- `Keyboard Analyzer` 查看各种键的快捷键
- `Setting Search` ob 的原生搜素不全面，这个设置搜索好用
- `Style Settings` 对于主题的 css 设置
- `Linter` 格式化文档
- `chat view` 聊天气泡
- `Timeline` 时间轴渲染
